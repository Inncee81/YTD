YTD (YouTube Downloader)

Windows a Linux verze free windows YouTube Downloaderu YTD, klient pro stahování videa nejen z YouTube, ale i z dalších serverů.

Popis aplikace

Program pro stahování video souborů z internetu

Více o fungování programu na stránkách autora http://www.pepak.net/ytd/ytd

Stabilní funkce:

  • přidání URL ze schránky
  • načtení seznamu URL ze souboru (txt,html)
  • uložení seznamu URL do souboru
  • start, stop stahování
  • nastavení konfigurace
  • protokoly http, https
  • stahování RTMP
  • podpora MSDL (mmst,mmsh,real-rtsp,wms-rtsp)

Testovací prostředí
